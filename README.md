###### **接口说明：**

<h3 id="1">**一、导航**</h3>
<h4 id="1.1">**#11获取导航栏信息**</h4>
接口说明：
>获取小程序应用列表的首页导航
>请求参数：

| 参数名       | 含义       |           规则说明           | 参数类型    | 是否必须 | 缺省值|
| ------------ | --------   | ---------------------------- | ----------- | -------  | ----  |
| token        |登录凭证    |用户认证令牌                  |  integer    | 是       | 无    |

请求实例：
   GET
   [cate/index]()
   HTTP/1.1 Host: [http://web.server.com:8080/api/]

返回结果：
登录成功

```json
{
    "code": 0,
    "message": "获取导航栏信息成功",
    "data": [
        {
            "id": 1,
            "name": "移动电商",
            "sort": 1
        },
        {
            "id": 2,
            "name": "本地O2O",
            "sort": 2
        },
        {
            "id": 3,
            "name": "智慧门店",
            "sort": 3
        }
    ]
}
```
返回参数：

| 参数名       | 含义              | 参数类型   | 长度 |
| ------------ | ------------------| ---------- | ---- |
|id            | 导航栏表主键ID    | integer    |  -   |
|name          | 导航栏名称        | string     |  -   |
|sort          | 排序，越小越靠前  | integer    |  -   |
